Using D3.js Roadmap

Future Work

In the summer of 2019 as part of a grant from Bridgewater College we'll be continuing our work and add chapters on the following modules.
  • Paths (render to SVG code written for 2D canvas)
  • Contours (computes contour polygon, often passed to Path)
  • Polygons (operations on 2D polygons)
  • Quadtrees (partitions and models 2D space into squares)

  • Dispatches (used to register and call listeners for custom events)
  • Dragging (for mouse or touch input)
  • Brushes (select a 1D or 2D region using mouse or touch input)
  • Zooming (Pan and zoom using mouse or touch input

  • Transitions (animated transitions for selections)
  • Easings (functions for smooth animations)
  • Timers (queue for managing concurrent animations)

  • Interpolators (interpolation methods for blending between two values - referenced in 04-05)

  • Collections (Objects, Maps, Sets, Nests)
  • Arrays(Statistics, Search, Transformations, Histograms)

  • Delimiter-Separated Values (conversions between DSV and JSON - added to 03 )
  • Colors (conversion from CSS to RGB or HSL)
  • Number Formats
  • Time Formats
  • Time Intervals (used by d3's time scales to generate ticks - referenced in 04-04)
  • Random Numbers
